## 🏃 如何啟動

### 方法一: 直接使用 (推薦)

```bash
# 1. 安裝Flask
pip install flask --break-system-packages

# 2. 啟動系統
python3 app.py
```

### 方法二: 重建資料庫

```bash
# 1. 安裝相依套件
pip install flask pandas openpyxl xlrd --break-system-packages

# 2. 重建資料庫 (如果需要)
python3 create_database.py

# 3. 啟動系統
python3 app.py
```
